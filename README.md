# GIOC

A go IOC container.


## Authors

- [@xiaocailc](https://github.com/xcocx)


## Installation

### Prerequisites

- **[Go](https://go.dev/)** >= 1.20

With [Go module](https://github.com/golang/go/wiki/Modules) support, simply add the following import

```bash
import "github.com/hugiot/gioc"
```
